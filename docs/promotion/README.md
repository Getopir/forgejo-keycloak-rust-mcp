# Promotion Checklist

This folder contains public promotion and project-readiness material for `forgejo-keycloak-rust-mcp`.

## Codeberg Release

Create a Codeberg release for `v2.0.0` using the text in [codeberg-release-v2.0.0.md](codeberg-release-v2.0.0.md).

Codeberg releases are separate from Git tags. The `v2.0.0` tag should be pushed first, then the hosted release object and its signed source and SBOM assets must be created through the Codeberg frontend or API.

## Codeberg Issues

The original pre-1.0 issue drafts are archived. Create new public issues from
the current [maintainer backlog](../wiki/Further-Planned-Improvements.md).

## OpenSSF Best Practices

The verified OpenSSF Passing badge and follow-up criteria are recorded in [openssf-best-practices-checklist.md](openssf-best-practices-checklist.md).
