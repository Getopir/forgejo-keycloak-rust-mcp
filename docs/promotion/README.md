# Promotion Checklist

This folder contains public promotion and project-readiness material for `forgejo-keycloak-rust-mcp`.

## Codeberg Release

Create a Codeberg release for `v1.2.3` using the text in [codeberg-release-v1.2.3.md](codeberg-release-v1.2.3.md).

Codeberg releases are separate from Git tags. The `v1.2.3` tag should be pushed first, then the hosted release object must be created through the Codeberg frontend or API.

## Codeberg Issues

The original pre-1.0 issue drafts are archived. Create new public issues from
the current [maintainer backlog](../wiki/Further-Planned-Improvements.md).
