# Codeberg Release Text - v1.0.2

Title: `v1.0.2 - Crates.io publication metadata`

`v1.0.2` is a metadata-only publication patch for crates.io.

## Highlights

- Aligns crate metadata with the public `GetOpir` Codeberg organization.
- Keeps the same hardened MCP gateway behavior as `v1.0.1`.
- Provides a matching git tag for the commit that should be published to
  crates.io.

## Install

```sh
cargo install forgejo-keycloak-rust-mcp --version 1.0.2 --locked
```
