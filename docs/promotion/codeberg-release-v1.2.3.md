# Codeberg Release Text - v1.2.3

Title: `v1.2.3 - open contribution guidance for humans and AI agents`

`v1.2.3` simplifies the contributor workflow and explicitly welcomes reviewed
AI-assisted contributions. The release does not expand the MCP gateway's
executable Forgejo capability surface.

## Highlights

- Simple contributor steps and verification requirements.
- Root `AGENTS.md` guidance for AI coding agents.
- Clear contributor responsibility for review, licensing, and tests.
- Matching hosted wiki contribution page.
- Signed source archives with SHA-256 checksums and SSH verification guidance.

## Install

```sh
cargo install forgejo-keycloak-rust-mcp --version 1.2.3 --locked
```
