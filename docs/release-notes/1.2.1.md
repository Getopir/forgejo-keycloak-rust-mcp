# Release Notes - 1.2.1

`1.2.1` adds bounded pull-request diff inspection and evidence-backed review submission for governed agent workflows.

Highlights:

- Adds `get_pull_request_diff`, returning bounded pull-request metadata, changed-file summaries, and diff text without exposing downstream credentials.
- Enforces server-side byte, file, line, and pagination caps so independent review remains useful without unbounded context growth.
- Adds `submit_pull_request_review`, allowing mapped Forgejo principals to submit approve or request-changes reviews with evidence in the review body.
- Adds matching `forgejo-mcpctl` commands, policy classifications, capability discovery, tests, and operator documentation.

Install:

```sh
cargo install forgejo-keycloak-rust-mcp --version 1.2.1 --locked
```
