# Release 1.2.9

## Changes

- Runs `cargo clippy --workspace --all-targets -- -D warnings` in both internal
  Forgejo and public Codeberg CI.
- Applies strict source analysis on every push and pull request.
- Treats every Clippy finding as a CI failure before tests and release checks
  can complete.
- Updates OpenSSF evidence and the maintainer backlog for the
  `warnings_strict` and `static_analysis_often` criteria.

No publicly known vulnerability was fixed in this release.

## Install

```sh
cargo install forgejo-keycloak-rust-mcp --version 1.2.9 --locked
```
