# Release 0.4.1

`0.4.1` is a beta-4 patch release that changes the project license to a stronger copyleft license.

## Changed

- Relicensed the project from `MIT OR Apache-2.0` to `AGPL-3.0-or-later`.
- Replaced the dual permissive license files with the GNU Affero General Public License v3.0-or-later text.
- Updated SPDX identifiers, crate metadata, README, contribution guidance, wiki source, and Codeberg publishing checklist.

## Rationale

The gateway is network server software. `AGPL-3.0-or-later` keeps modified network deployments open by requiring operators of modified versions to offer corresponding source to users who interact with the software over a network.
