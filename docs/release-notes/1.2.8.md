# Release 1.2.8

## Changes

- Publishes private vulnerability-reporting instructions using the monitored
  `info@getopir.com` project role address instead of a maintainer's personal
  email address.
- Commits to acknowledging reports within seven calendar days and providing
  material status updates at least every 14 calendar days while a case is open.
- Documents restricted access, redacted internal coordination, encrypted
  transfer of sensitive evidence, coordinated disclosure, and retention.
- Updates issue routing, OpenSSF evidence, and the source-controlled wiki to
  direct security reports away from public issues.

No publicly known vulnerability was fixed in this release.

## Install

```sh
cargo install forgejo-keycloak-rust-mcp --version 1.2.8 --locked
```
