# Release 0.4.0

`0.4.0` is the public beta-4 baseline release.

## Validation

- Built `forgejo-mcpd` in release mode.
- Tested the live Keycloak issuer with a full-scope agent and a read-only agent.
- Verified all registered Phase 0 operations return the expected policy decisions.
- Verified read-only tokens are denied for write, merge, and delete operations.
- Verified missing bearer tokens, invalid authorization schemes, invalid JWTs, and unknown operations are rejected.
- Verified no raw tokens or client secrets were printed in test output.

## MCP Surface

- `GET /health`
- `GET /.well-known/oauth-protected-resource`
- `GET /.well-known/oauth-protected-resource/mcp`
- `POST /mcp`

## Registered Operations

- `gateway_probe`
- `list_repository_metadata`
- `create_issue_comment`
- `merge_pull_request`
- `delete_repository`

## Limitations

This release still performs identity validation and operation-class policy decisions only. Forgejo API execution, Keycloak-to-Forgejo principal mapping, and persistent approval records remain planned follow-up work.
