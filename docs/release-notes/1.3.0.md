# Release 1.3.0

## Changes

- Adds the verified OpenSSF Best Practices Passing badge for project `13642`.
- Records the externally assigned project ID, earned badge level, and evidence URL.
- Updates the maintained source documentation and wiki to identify `1.3.0` as
  the current public-assurance release.
- Retains the executable Forgejo capability surface and security controls from
  `1.2.11` without compatibility-breaking API changes.

No publicly known vulnerability was fixed in this release.

## Install

```sh
cargo install forgejo-keycloak-rust-mcp --version 1.3.0 --locked
```
