# Release Notes - 1.2.2

`1.2.2` is a maintenance and operational-hardening release. It does not add
new executable Forgejo operations.

Highlights:

- Completes REUSE 3.3 copyright and license metadata for maintained files and
  records the separate MIT attribution for vendored Forgejo API data.
- Adds credential-rotation procedures for mapped Forgejo tokens, Keycloak
  clients and signing keys, and release credentials.
- Adds an incident-response procedure covering containment, evidence,
  credential replacement, recovery verification, and follow-up.
- Updates the source-controlled wiki and maintainer backlog to reflect the
  completed work.

Install:

```sh
cargo install forgejo-keycloak-rust-mcp --version 1.2.2 --locked
```
