# Release 0.3.15

`0.3.15` is the public beta-3 fix-15 release.

## Included

- Keycloak OIDC discovery and JWKS-based token validation.
- Strict bearer header parsing.
- Audience validation for string and array `aud` claims.
- OAuth protected-resource metadata endpoints.
- Phase 0 `/mcp` policy probe.
- Operation registry with risk class, required scope, and approval requirement.
- Structured audit event model that avoids token and secret values.
- Public install, configuration, agent, testing, and lab documentation.

## Not Included Yet

- Forgejo API execution.
- Persistent Keycloak-to-Forgejo principal mapping.
- Trusted-header Forgejo delegation.
- Generated full Forgejo API coverage.
- Built-in Keycloak client provisioning.

## Upgrade Notes

This is the first public-ready release line. Do not publish older internal snapshots.
