# Release 1.2.11

## Changes

- Adds reproducible stable-Rust coverage measurement with pinned
  `cargo-llvm-cov` `0.8.7`.
- Enforces at least 55% line coverage, 50% function coverage, and 55% region
  coverage across domain code in internal Forgejo and public Codeberg CI.
- Publishes the measured baseline of 59.62% lines, 52.48% functions, and 58.50%
  regions in test documentation and the hosted-wiki source.
- Documents the narrow entrypoint exclusion and requires review and rationale
  before thresholds or scope can be weakened.
- Updates OpenSSF evidence and the maintainer backlog for `test_most`.

No publicly known vulnerability was fixed in this release.

## Install

```sh
cargo install forgejo-keycloak-rust-mcp --version 1.2.11 --locked
```
