# Release 1.0.0

`1.0.0` is the first stable public release of the Forgejo Keycloak Rust MCP
gateway.

## Stable Features

- Keycloak bearer-token validation with issuer, audience, expiry, and JWKS.
- OAuth protected-resource metadata for MCP clients.
- Explicit policy registry for every exposed operation.
- Keycloak-to-Forgejo principal mapping with duplicate, disabled, and malformed
  mapping rejection.
- Trusted-header spoof rejection.
- Bounded repository, issue, pull-request, review, release, notification, and
  generated coverage readbacks.
- Stable `forgejo://...` resource URIs.
- File-backed, short-lived, single-use approval records.
- Approval-backed pull-request merge and release creation.
- Generated Forgejo API classification coverage from the pinned
  `15.0.3+gitea-1.22.0` Swagger document.
- `forgejo-mcpctl` CLI wrapper for humans and agents.

## Hardening Since 0.11.0

- OAuth protected-resource metadata now returns unique scopes.
- `forgejo_api_coverage` supports an explicit `query` field while preserving
  old `target` search fallback.
- Public docs and wiki now describe shipped features first instead of treating
  completed phases as future roadmap.
