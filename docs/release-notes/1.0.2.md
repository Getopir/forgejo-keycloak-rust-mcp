# Release Notes - 1.0.2

`1.0.2` is a crates.io publication metadata patch.

## Changed

- Updates all workspace crate versions to `1.0.2`.
- Aligns published crate metadata with the public Codeberg organization:
  `https://codeberg.org/GetOpir/forgejo-keycloak-rust-mcp`.
- Adds a matching release tag target for the commit that contains the current
  Codeberg repository, homepage, and documentation links.

## Compatibility

This release has no intended runtime behavior changes from `1.0.1`.

## Install

After publication:

```sh
cargo install forgejo-keycloak-rust-mcp --version 1.0.2 --locked
```
