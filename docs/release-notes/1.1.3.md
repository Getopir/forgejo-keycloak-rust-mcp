# Release Notes - 1.1.3

`1.1.3` expands the curated Forgejo MCP surface for governed agent work.

Highlights:

- Adds additive `create_issue` support through mapped Forgejo principals.
- Adds bounded wiki page listing and readback.
- Adds approval-backed wiki page creation and update with dry-run previews.
- Adds `credential_reference_status` for token-reference presence checks without returning secret values.
- Updates `forgejo-mcpctl`, policy registration, capability metadata, and public docs for the new operations.

Install:

```sh
cargo install forgejo-keycloak-rust-mcp --version 1.1.3 --locked
```
