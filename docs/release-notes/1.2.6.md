# Release Notes - 1.2.6

`1.2.6` completes four security and operational-hardening backlog items without expanding the executable Forgejo operation surface.

## Changes

- Publishes a maintained threat model covering assets, actors, trust boundaries, data flow, threats, controls, residual risks, assumptions, exclusions, deployment requirements, and review triggers.
- Adds checksum-verified Gitleaks scanning of complete pull-request and push history before Rust CI checks.
- Adds optional append-only structured audit export through `FORGEJO_MCPD_AUDIT_LOG`, with per-record flush and filesystem synchronization.
- Documents the startup-only JWKS snapshot, missing TTL and size limits, unknown-`kid` behavior, and the required overlap-and-restart key-rotation procedure.
- Updates the public security policy, wiki navigation, current-release documentation, and maintainer backlog.

## Install

```sh
cargo install forgejo-keycloak-rust-mcp --version 1.2.6 --locked
```

No MCP operation, scope, approval requirement, or Forgejo capability was added in this release.
