# Release Notes - 1.2.4

`1.2.4` completes three release-engineering and repository-governance backlog items without changing the MCP operation surface.

## Changes

- Validates CycloneDX document generation in CI.
- Regenerates and attaches workspace SBOMs when a Forgejo release is published.
- Includes all release SBOMs in the signed `SHA256SUMS` manifest.
- Runs Renovate weekly against the internal Forgejo repository with automerge disabled.
- Runs the existing pull-request checks on every dependency update before maintainer review.
- Adds the public CI check needed before Codeberg `main` can enforce required review and status checks.

## Install

```sh
cargo install forgejo-keycloak-rust-mcp --version 1.2.4 --locked
```

No identity, policy, approval, or executable Forgejo operation behavior changed in this release.
