# Release Notes - 1.1.0

`1.1.0` adds branch-to-pull-request workflow support and operator-friendly capability discovery.

## Added

- `create_pull_request` MCP operation.
- `forgejo:pr:write` policy scope.
- Approval-gated PR creation with dry-run preview.
- Optional PR body, draft flag, assignee, assignees, and reviewer request inputs.
- Reviewer request status and error reporting after PR creation.
- `forgejo-mcpctl create-pull-request` CLI command.
- `GET /capabilities` endpoint listing operation names, scopes, risk classes, approval requirements, and planned disabled operations.

## Documentation

- Adds agent guidance for short-lived Keycloak token broker flows.
- Documents canonical issuer/resource URL requirements for operators.
- Updates the wiki roadmap and planned-improvement pages to mark PR creation and operation discovery as shipped.
- Regenerates Forgejo API coverage with 10 reviewed semantic overlays and 481 disabled metadata-only operations.

## Still Disabled

- Standalone PR update.
- Standalone reviewer request.
- Branch status readback.
- Required-check readback.
- PR-check readback.
- Generic generated endpoint execution.
- Admin and destructive execution.
