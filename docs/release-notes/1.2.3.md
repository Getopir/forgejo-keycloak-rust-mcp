# Release Notes - 1.2.3

`1.2.3` is a documentation and contributor-experience release. It does not add
new executable Forgejo operations.

Highlights:

- Simplifies the public contribution guide.
- Explicitly welcomes reviewed AI-generated and AI-assisted contributions.
- Adds a root `AGENTS.md` with repository constraints and required checks for
  coding agents.
- Publishes matching contribution guidance in the hosted wiki.
- Publishes signed source archives, SHA-256 checksums, and verification guidance.

Install:

```sh
cargo install forgejo-keycloak-rust-mcp --version 1.2.3 --locked
```
