# Release Notes - 1.0.1

`1.0.1` is a packaging release for crates.io distribution.

## Added

- Prepared the workspace for crates.io publication with unique package names:
  - `forgejo-keycloak-mcp-policy`
  - `forgejo-keycloak-mcp-identity`
  - `forgejo-keycloak-mcp-audit`
  - `forgejo-keycloak-rust-mcp`
- Added explicit installable binary names:
  - `forgejo-keycloak-rust-mcpd`
  - `forgejo-mcpctl`
- Added crates.io metadata for repository, homepage, documentation, keywords,
  categories, description, license, and Rust version.
- Added `docs/crates-io-publishing.md` with publish order and release checklist.
- Updated install docs to show:

```sh
cargo install forgejo-keycloak-rust-mcp --locked
```

## Changed

- Renamed publish package names away from generic `audit`, `identity`, and
  `policy` so they are suitable for the global crates.io namespace.
- Kept short Rust library crate names for source compatibility.

## Security

- No runtime secret values are included in crate metadata or install docs.
- The license remains `AGPL-3.0-or-later`.
