# Forgejo Keycloak Rust MCP

This wiki explains the public `forgejo-keycloak-rust-mcp` project for humans and agents.

## Purpose

The gateway lets agents prove identity through Keycloak before receiving policy decisions for Forgejo-related operations. Forgejo remains the final authority for repository and organization access.
